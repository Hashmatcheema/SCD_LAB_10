# Node.js Event Planner Application

This is a Node.js application for event planning and management.

## Features

- Create events
- View events
- Event validation

## Installation

```bash
npm install
```

## Running Tests

```bash
npm test
```

## Jenkins CI/CD

This project includes a Jenkins pipeline for continuous integration.

### Webhook Configuration

The pipeline can be automatically triggered via GitHub webhooks. See `WEBHOOK_SETUP.md` for detailed instructions.
